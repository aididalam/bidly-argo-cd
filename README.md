Root project: [bidly](https://github.com/aididalam/bidly)

## Demo marketplace data

`bidly-demo-seed` runs after the API deployments are healthy and creates five idempotent demo accounts (`user1@bidly.com` through `user5@bidly.com`, password `password`), twenty active listings, and ten bids. It is intended for the sandbox environment only; replace or remove these credentials before production use.
